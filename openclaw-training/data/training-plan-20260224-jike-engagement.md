## 五、指令模板（发送给 Agent 的完整消息）

### T1-1 指令

```

4. 回复我：工具执行成功 + 3 条动态摘要

交付物：飞书回复中包含 3 条动态摘要
```

### T1-2 指令

```
【培训任务 T1-2】圈子动态调研

请执行以下操作：
1. 用 search-topic 搜索「AI」或「AI探索站」，获取圈子 ID
2. 用 topic-feed <圈子ID> 浏览该圈子内的动态
3. 选取 3 条你认为「互动较高」或「写法有特色」的动态，分析其特征
4. 创建文件：content/workspace/jike-engagement-research/explore-notes.md
   内容需包含：每条动态的标题/开头、结构、话题、互动数据、可借鉴点
5. 回复我：文件已创建 + 3 条动态的核心可借鉴点（一句话每条）

交付物：explore-notes.md 文件，且回复中包含可借鉴点总结
```

### T2-1 指令

```
【培训任务 T2-1】高互动性即刻文章创作

基于你在 T1 中的调研结论，请完成：

1. 创建创作文档：content/workspace/jike-engagement-research/draft-engagement.md
   - 选题理由（为什么选这个主题、预期互动点）
   - 完整动态内容（可直接复制到 jike-toolkit 的 post 命令）
   - 配图说明（如需配图：图片主题、文字内容、使用 gen_cover.py 时需指定中文字体）

2. 使用 jike-toolkit 发布到即刻：
   - 纯文字：node jike-toolkit.mjs post "内容"
   - 带图：node jike-toolkit.mjs post "内容" --images /path/to/img.jpg
   - 发到圈子：node jike-toolkit.mjs post-topic <圈子ID> "内容" [--images ...]

3. 回复我：帖子 ID、链接、发布成功确认

铁律提醒：配图必须使用中文字体（见 TOOLS.md gen_cover.py 规范），发帖必须用 jike-toolkit，禁止用其他方式。
```

