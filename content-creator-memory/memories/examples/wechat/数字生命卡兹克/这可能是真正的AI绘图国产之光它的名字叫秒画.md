# 这可能是真正的AI绘图国产之光，它的名字，叫秒画。

原创

数字生命卡兹克

数字生命卡兹克

*2024年07月23日 12:08*
*北京*

最近很多人在问，有没有啥国产的比较好的AI绘图产品。

比如这个朋友。

![](./assets/17516869224610.31693821275269285.jpeg)

如果你非要我推一个AI绘图产品，那没啥可说的，必然是Midjourney，这玩意属于AI绘图界的OpenAI，属于断层式领先。

但是很多人在问的时候，会加一个定语：**国内**。

逻辑其实挺简单的，有太多普通人，只是为了画个图而已，就是简简单单的画个图。

比如上面的那位朋友，他就是一位老师，他想用AI绘图，可能也就是给课件配个精美的插图，仅此而已。

MJ好是好，但是需要魔法，需要很贵的金额，光这两点，其实就能劝退无数人了。

在AI越来越普通人化的情况下，AI产品的国产化，也越来越势不可挡。

像liblib那种在线SD做的也挺好，但是普通人用起来还是成本太高，选项一堆，而且那么多模型，我到底该选哪个？

![](./assets/17516869235630.03727130298903292.png)

所以我们需要的，其实是国产的类似于Midjourney那样的大模型，一站式解决，不用切来切去，烧脑到极致。

于是我们就花了两天时间，把国产AI绘图大模型全都跑了一遍。

![](./assets/17516869239130.967994065182041.png)

至于具体的详细对比文档，我就不放了，容易得罪人。

但是结果还是有趣的，因为我确实发现一个国产模型，是真的能跟MJ掰掰手腕的。

它的名字，叫：**秒画**。来自商汤。

![](./assets/17516869223530.749370552406929.png)

网址在此：https://miaohua.sensetime.com/

非常的低调，我搜了下，基本没做过任何宣传。

但是跑出来的图片说实话让我非常惊艳，他们的 Artist v1.0 Alpha模型，甚至有的领域能比肩MJ6.0，不像有些吹自己超过MJ多少多少的，一跑起来啥也不是= =

而且，**20s以内，就能直接生成8张图**，这个速度也是非常的恐怖。。

最关键的是，现在还不要钱，免费能用。反正用的时候，就让朋友点一下你的链接，就能白嫖3天无限卡。

![](./assets/17516869230490.7479724971957935.png)

我直接放跟MJ的对比图，这样直观，你就能感受到，这玩意有多强了。

我分四个领域给大家对比一下，分别是**：人像摄影、写实、3D、2D**。

**一.人像摄影**

1. 特写，Elle 杂志，美女，可口可乐药店，20 世纪 90 年代街头服饰，入口处的华丽女人，柯达 Porta 800 胶片，模拟，哈苏，明亮而大胆。

![](./assets/17516869233550.24534960519255067.jpeg)

2. 我们第一次约会时迷人的韩流女孩的手机照片自拍，户外，晚上时间，发布在 Snapchat 上，黑发，休闲傻笑，2 个发髻绑发型。

![](./assets/17516869233440.5547990736655383.jpeg)

3. 摄影，一位有影响力的人与她的朋友在客厅的自拍照。她穿着一件黑色皮夹克，棕色的头发向后扎成一个紧紧的马尾辫。他们都有黑眼睛。左边的女人戴着珍珠耳环。

![](./assets/17516869227600.9581488500064815.jpeg)

4. 一个穿着蓝色衬衫和短裤的英俊男人，在海滩上，阳光明媚的蓝天，走向镜头。他身穿蓝色火烈鸟印花图案衬衫，浅蓝色裤子，戴着时尚摄影风格的墨镜。

![](./assets/17516869233580.7264180226459643.jpeg)

5. 一位刚结束长期太空任务的宇航员。他坐着,仍穿着航天服,只是摘下了头盔。他的脸上显露疲惫和惊奇,明显可见失重环境造成的身体影响。航天服因任务而磨损和留有痕迹,可见生命支持系统的复杂细节。洁白的隔离室与磨损的航天服形成鲜明对比。附近桌上放着一个容器中的小植物,这是他与地球生命的首次重逢。使用中画幅数码相机拍摄,临床照明强调纹理。色彩处理突出冷色调的蓝色和白色,植物的绿色是唯一的暖色元素。

![](./assets/17516869222950.5142277340637517.jpeg)

![](./assets/17516869234640.04973262610778417.jpeg)

**写在最后**

相信大家整体看下来，是有自己心里的一杆秤的。

最难能可贵的，秒画除了细节之外，能真正做到，在审美、光影比肩MJ，人物摄影和写实还有一定的差距，但是在语义理解、3D、风格化等等，都能快打平了。

当然，说超越还为时过早，毕竟Midjourney那么多年的积累，真不是吹的。国产大模型想要超越，还得再努力很久，但是秒画能在某些领域能对标MJ，能比肩MJ，而且是没搞一些乱七八糟的东西，扎扎实实做产品，这个点，我就觉得已经非常非常好了。

对于国内真的认真做产品的公司，我愿意追踪，愿意给他们很大的包容度，也愿意给予最大的耐心，陪伴他们成长。

因为说不定。

他们，就是下一个。

AI时代的，超级巨星。
