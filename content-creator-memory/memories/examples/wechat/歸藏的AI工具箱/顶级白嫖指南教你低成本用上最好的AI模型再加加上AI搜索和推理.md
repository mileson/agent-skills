模型白嫖方案
------

我这套方案目前的主力模型是两套，分别是 DeepSeek R1 和 V3 以及 Gemini2.0 Flash Thinking。　

Deepseek R1 完成文案创作和一些需要复杂推理的任务，V3 完成一些基础任务，比如总结信息翻译等。　

Gemini 用来补充 Deepseek R1 没有的多模态推理能力，Gemini 2.0 的视频和音频多模态理解能力也是现在最好的。　

刚好这两个模型都可以完全的白嫖。　

Deepseek R1 和 V3 可以参加火山的邀请活动，直接获得 45 块钱的 Token 额度加上免费的 50 万都够用很久了，同时他们的 Deepseek 模型速度和质量都应该是现在最好的，更是把 R1 的输出长度拉到了 16K。　

链接在这里：https://www.volcengine.com/activity/deepseek　

Gemini 的话谷歌更是大方，个人使用完全可以免费，没有任何套路，你只需要去 https://aistudio.google.com/ 左上角创建一个 API 就行，直接可以用 2.0 所有的模型，最起码我最近使用没有遇到任何限流。　

除了这两个以外最近还有一些可以白嫖 API 额度的活动我也都放在这里你可以领。　

首先是可以在 XAI 的 API 平台领取 150 美元额度，等个一两周 Grok 3 的 API 开放可以用，领取方式是登录 XAI 后台https://console.x.ai/team/ 之后选择在下面位置的按钮点击共享使用数据就行。　

另外 Elevenlabs 搞了一个赞助 AI 开发者的活动里面也有比如欧洲顶尖 AI 模型 Mistral AI 的 25 美元API 额度，可以在这里领取：https://www.aiengineerpack.com/?success=1

如何使用这些 API 和模型
--------------

我们现在手里已经有很多 AI 服务平台白嫖来的 API 了，那么应该如何使用呢。　

这里我主要介绍我用的 AI 客户端 ChatWise https://chatwise.app/ ，他的设置非常简单而且有很多很强大的功能，尤其支持完全免费的 AI 搜索能力，而且搜索结果质量甚至比 Perplexity 还要高很多。　

这个软件基础的聊天能力是免费的，搜索功能需要付费，如果你觉得有需要，可以用藏师傅的优惠码「guizang」填写之后会有 8 折优惠，而且这个不是订阅制，是买断制。　

你也可以用比如 Cherry Studio （https://cherry-ai.com/）这种免费客户端开源也是一样的，配置过程都类似也可以参考。　

接下来我们以 ChatWise 为例子说一下如何配置模型的 API，这里主要讲两个一个是火山的 API 和 Gemini 的。　

如果要在 ChatWise 配置模型需要在软件左下角点击设置，然后进到模型设置页面，之后点击左下角的「➕」号，在里面找到 Volcengine。　

可以发现模型名字和地址都给我们写好了，我们只需要去火山的后台找到 API Key 就行，直接访问这个链接 https://console.volcengine.com/ark/ 然后点击Deepseek V3 和 R1 的开通服务。然后点击下面的 API Key 管理复制我们的 API Key 填入刚才 ChatWise 的输入框就行。　

好了处理完最复杂的一个了，剩下的就简单了，你刚才应该已经从 AI studio 获得了创建的谷歌 Gemini 的 API Key 了。只需要在模型设置这里将 API Key 贴进去就行。　

其他的模型都是类似的处理，比如三方的 open AI 或者 claude 模型需要自定义 API 的 URL 一般，平台都在最API 设置页面最显眼的部分写了 API 地址，更改一下就行，如果你是官方的 API 的话就用默认地址就行。

为模型接入搜索、Artifacts和推理
--------------------

我们接入模型之后除了基本的聊天之外一般还需要一些高级功能，比如需要让模型联网搜索应该搜多的信息，或者让模型输出的代码可以被渲染为网页，就是常说 Artifact 功能。　

ChatWise 的联网是我见过最强大最便宜的，其他的软件实现联网都需要一个模型 API 才行，他直接用你本地的浏览器搜索结果，完全免费不说用的还是谷歌的搜索结果，比其他家强大。　

首先你在聊天界面输入框这里，点击地球图标就能启用搜索。　

他在搜索这里会有一些非常牛批的功能和设置，我来介绍一下。　

*   首先是搜索引擎这里我们选择谷歌这里就是你本地的谷歌浏览器搜索结果不需要付费。
*   然后是用于生成搜索关键词的模型，这里我推荐火山的 Deepseek V3 上下文长，而且很便宜，Chatwise 会用这个模型帮你生成搜索词和总结网页的搜索结果，然后在把润色后的结果给你当时选择的模型参考回答。
*   然后是搜索关键词语言，我们简中互联网的信息质量大家都知道的，ChatWise 可以指定搜索的结果，比如在学术内容的搜索上只搜索英文内容。
*   最后一个更骚的，可以屏蔽指定网站的搜索结果，这个的好处是，比如 CSDN 这种网站 AI 是没办法读取任何信息的。然后你设置了 10 个网页的搜索上限，结果里面 6 个是 CSDN，实际上就只搜了 4 个。

然后是 Claude 的看家功能 Artifact，可以将 AI 输出的大段文档和代码在另一个面板显示，而且可以将代码渲染成网页或者图表。　

只需要点击输入框叶子图标就可以自动启用，可以看到图面里面 Artifacts 甚至还帮我画了 mermaid 流程图。　

最后一个 ChatWise 的强大功能是可以为，没有推理能力的模型加上推理能力，实现的逻辑是将 Deepseek R1 的搜索过程发给这个模型，由于这个时候的输出 Token 大小是 1 所以用官方的 API 的话生成推理过程甚至也是不花钱的！
