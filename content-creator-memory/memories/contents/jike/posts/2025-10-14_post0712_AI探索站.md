# 即刻动态 712

**发布时间:** 2025-10-14  
**圈子:** AI探索站

---

Karpathy 发布了一个全新的开源项目 nanochat，它是一个从零开始、端到端的 ChatGPT 简化版训练与推理框架。相比他之前的 nanoGPT（只涵盖预训练部分），这次 nanochat 是一个完整的可运行系统，目标是：你只需在云 GPU 上运行一个脚本，约 4 小时后就能在一个 ChatGPT 风格的网页界面中与自己的 LLM 对话。整个代码库仅约 8,000 行，干净、轻量、可读性强。

它的主要功能包括：
	• 使用 Rust 实现的新 tokenizer 进行训练
	• 在 FineWeb 数据集上预训练 Transformer LLM，并通过 CORE 指标评估
	• 进行中期训练（Midtrain），包括用户-助手对话、选择题、工具使用等
	• SFT 阶段评估模型在多项任务的表现，例如世界知识题（ARC-E/C、MMLU）、数学（GSM8K）、代码（HumanEval）
	• 可选使用“GRPO”方法在 GSM8K 上进行强化学习（RL）
	• 推理阶段使用高效 Engine（带 KV cache、prefill/decode、Python 沙盒工具调用），支持 CLI 与 WebUI 交互
	• 自动生成 Markdown 格式的训练报告卡（report card），让训练过程可视化、游戏化

从成本来看，只需约 $100（在 8xH100 GPU 节点上训练 4 小时），你就能得到一个能写诗、讲故事、回答基础问题的小型 ChatGPT 克隆；约 12 小时的训练表现即可超越 GPT-2，在 CORE 指标上更优。若扩展至 $1000（约 41.6 小时训练），模型将更连贯，能解决简单的数学和代码问题，并在 MMLU、ARC-Easy、GSM8K 等基准上取得不错成绩（例如 MMLU 40+、ARC-Easy 70+、GSM8K 20+）。

Karpathy 的目标是打造一个「最小、可读、可黑、易复刻」的强基线（strong baseline）栈，用于教学课程 LLM101n，并可能发展成研究用 benchmark 或训练框架。当前版本虽未完全优化，但整体框架已稳定，适合开源社区继续改进。

如果你想亲手跑一个可聊天的轻量模型，或者学习 LLM 从预训练到推理的全流程，nanochat 非常值得尝试。
