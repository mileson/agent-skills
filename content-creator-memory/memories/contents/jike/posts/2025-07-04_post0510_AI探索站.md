# 即刻动态 510

**发布时间:** 2025-07-04  
**圈子:** AI探索站

---

Cursor 1.2 版本最难用的功能，这里有个解决办法！

这两天更新了 Cursor 新版本 1.2 后，发现聊天窗口栏目变成折叠起来了，而且每次切换不同聊天窗口都需要非常小心，因为关闭按钮跟点击按钮重叠在一起了！
	
在小心翼翼尝试了好几次后，我几乎崩溃了… 真的是绝处逢生，我发现了个办法可以恢复原先的长方形切换按钮的办法(｡•̀д•́)✧👇
1️⃣ 鼠标右击聊天窗口切换按钮
2️⃣ 在「Activity Bar Position」 处把「Default」这个选项勾选起来（默认是没有勾选的）
3️⃣ 你就可以看到聊天窗口切换按钮变回来啦[哭惹R]
	
希望对大家有帮助，尤其是经常开多个窗口的创造者们，这次 Cursor 更新真的给我造成不少麻烦，在没有新版本之前只能先这么配置了！
