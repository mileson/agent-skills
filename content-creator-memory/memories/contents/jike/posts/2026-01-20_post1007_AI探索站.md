> 一开始 vibe coding 的时候，我是直接把文字需求丢给 Ai，Coding Ai 虽然能把功能实现但经常抽出天际（如图一是调整多次后的，还是很丑）。
> 所以我还是用笨办法，专业的事情交给专业的 Ai 做，用上了 Figma Make 帮我先生成 UI。（图二）
> 用 Figma 生成的 UI ，可以导出代码，导出后放在项目目录下，让 Claude code 根据 UI 来生成产品，并在后续的迭代中需按照设计规范来。（图三）
> 
> 如果被纯 Ai Coding 出来的产品丑哭的，不妨可以参照这种方式。
