# 即刻动态 121

**发布时间:** 2025-01-12  
**圈子:** AI探索站

---

我还以为是因为项目复杂了导致的，原来不少人在使用 Cursor 的 Agent 模式的时候，也遇到了如下三个问题：

1、创建了原先已有的文件，已经确认做了索引，即记忆了已有文件。

2、执行不成功的情况，不断循环执行，还无法中断，这真的是让人崩溃的至暗时刻。

3、当执行命令回复内容过程中，重新编辑指令内容，已经回复的内容可能会串行乱码，很影响输入体验，担心可能还会出现不可控的改代码的情况（撤回还在回复的原因）。

希望Cursor 能重视，把这个核心的功能完善起来。

在此之前，如果遇到此类问题，建议大家通过 Restore 或者 Git 版本管理等方式进行回退，多次抽卡并且微调指令会得到一些缓解：）
