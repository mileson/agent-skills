# 即刻动态 431

**发布时间:** 2025-05-29  
**圈子:** AI探索站

---

LLM 产生幻觉，是因为它是通过预测下一个最可能出现的词来生成文本的，而不是通过理解事实或逻辑推理来回答问题。

1. 模型机制：语言建模本质是“填空题”
	• LLM 的核心任务是：在给定前文的情况下预测“下一个最可能的词”。
	• 它输出的是**“语言上最合理”而不是“事实最准确”**的结果。
	• 举例：问“谁发明了灯泡？”它可能答“爱迪生”，即使你想知道的是更具体的技术细节或其他相关人物如“约瑟夫·斯旺”。

2. 无事实校验：生成结果不经过事实验证
	• LLM 并不会自动查询外部数据库或知识库来确认答案是否正确。
	• 它只是从数以亿计的句子中“学会了模式”，但不会辨别“哪句是真的”。

3. 训练数据来源复杂，质量不一
	• 模型训练数据中混杂了网页、书籍、对话等各种来源的信息。
	• 其中很多信息可能是错误的、讽刺的、虚构的，模型很难分辨。
	• 比如它可能读过“乔布斯是宇航员”这样的段子，之后在不恰当的场合重复

4. 知识是“隐式记忆”，容易错配或拼接
	• 模型不是像数据库那样明确地“存储事实”，而是把语言模式压缩进参数中（“权重”）。
	• 当它试图“召回”知识时，可能会错误地组合多个片段。
	• 举例：你问一个冷门科学家的履历，它可能混合了多个类似人物的信息生成一个“合理但虚假的故事”。
