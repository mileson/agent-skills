## 特殊场景模板

### 报表页模板

```markdown
### X.3 功能页面
[图片：XX报表.png]

**筛选条件**：
| 筛选项 | 控件类型 | 说明 |
|-------|---------|------|

**功能按钮**：查询、导出、打印、返回

**报表字段**：
| 类别 | 字段名称 | 说明 |
|------|---------|------|
| 基础信息 | 字段A | 说明 |
| 数据分类1 | 字段B | 说明 |

**数据来源说明**：
- **字段X**：数据来源于【模块A】的XXX
- **字段Y**：数据来源于【模块B】中配置的XXX参数
- **字段Z**：系统根据X、Y自动计算
```

### 数据维度列模板（多期对比场景）

```markdown
**数据维度列**：
| 数据列 | 所属分组 | 说明 |
|--------|---------|------|
| 实际数 | 当期数据 | 从XX系统获取的实际数据 |
| 测算数 | 当期数据 | 当期测算数据 |
| 全年测算数 | 当期数据 | 全年测算数据 |
| 年度预算数 | 当期数据 | 年度预算数据 |
| 预算完成率 | 当期数据 | 预算完成比例 |
| 参考值 | 第二年数据 | 第二年参考值 |
| 调整值 | 第二年数据 | 第二年调整值 |
| 目标测算数 | 第二年数据 | 第二年目标测算数据 |
| 同比增减率 | - | 当期与第二年的同比增减比例 |
```

### 对比表模板

```markdown
**报表头信息**：
| 字段 | 说明 |
|------|------|
| 方案A | 原方案名称 |
| 方案B | 调整后方案名称 |

**对比指标**：
| 类型 | 指标名称 | 单位 | 说明 |
|------|---------|------|------|
| 业务指标 | 指标A | 头 | 说明 |
| 财务指标 | 指标B | 万元 | 说明 |

**数据展示维度**：
| 分组 | 数据列 | 说明 |
|------|--------|------|
| - | 实际数 | 历史实际数据 |
| 当期 | 调整后 | 调整后测算数据 |
| 当期 | 调整前 | 调整前测算数据 |
| 当期 | 对比 | 差异值（调整后-调整前） |
```
