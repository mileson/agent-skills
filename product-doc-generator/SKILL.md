# 产品功能说明书生成器

## 输入要素收集

开始前确认以下输入：

| 输入项 | 说明 | 必需 |
|--------|------|:----:|
| 参考文档 | 风格参考文档(.docx)，可设置权重比例 | 是 |
| 流程图 | 业务流程图(.png/.jpg) | 是 |
| 截图文件夹 | 命名规范：`01 功能名 - 页面类型.png` | 是 |
| 系统名称 | 如"生产排产系统" | 是 |
| 图片占位格式 | 默认`[图片：{filename}]` | 否 |

## 执行流程

```
1. 读取参考文档 → 提取风格（表格格式、描述语气、层级结构）
2. 读取流程图 → 识别业务节点和流程顺序
3. 扫描截图文件夹 → 按命名排序，识别功能模块
4. 逐一分析截图 → 提取字段信息（见下方提取规则）
5. 按模板生成文档 → 见 references/template.md
6. 用户反馈优化
```

## 截图字段提取规则

**列表页提取**：
- 查询条件：字段名、控件类型（文本框/下拉框/日期选择器/单选按钮组）、说明
- 功能按钮：按钮名称、功能说明
- 列表字段：序号、字段名、说明
- 操作列：操作名称、显示条件（如有）、说明

**新增/修改页提取**：
- 基本信息：字段名、是否必填、控件类型、说明
- 明细配置：按分类分组，列出所有配置项
- 计算规则：默认值来源、关联字段、计算公式

**报表页提取**：
- 筛选条件
- 数据维度列（分组归属）
- 汇总列说明
- 数据来源说明

## 质量检查清单

- [ ] 字段名称与截图完全一致（如"出库"vs"入库"）
- [ ] 数据来源和计算规则已说明
- [ ] 指标归属正确（区分不同品类/类型的指标）
- [ ] 数据维度列描述准确（跨年度对比字段）
- [ ] 各品类/模式的指标差异已体现
- [ ] 所有截图已分析完成

## 常见优化点

基于实践经验，以下内容容易遗漏，需特别注意：

1. **默认值计算逻辑**：字段默认值的来源和计算公式
2. **数据联动关系**：字段间的关联读取关系
3. **数据来源说明**：报表数据来自哪个模块
4. **指标归属**：同名指标在不同分类下的区别
5. **同比/环比字段**：跨期对比的计算基准说明
