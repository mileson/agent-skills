# Default Corporate Guidelines for Image Generation

This file defines a highly professional, default corporate visual identity optimized for business diagrams, architectural flows, and conceptual product illustrations. It is based on modern SaaS and Enterprise aesthetics.

## Core Identity
This style is clean, trustworthy, and structural. It relies on a high-contrast but visually soothing palette that clearly differentiates functional modules without being aggressive. It is perfect for flowcharts, data pipelines, and multi-tier architecture diagrams.

## Color Palette (To be injected into prompts)

When this brand is applied to image generation, use the following hex codes and color descriptions in the prompt:

**Primary Background & Canvas Colors:**
- `#fbfaf8` (Warm soft cream) - Primary light background, providing a paper-like, less straining canvas.
- `#f4f5f7` (Very light cool gray) - Use for secondary bounding boxes or card backgrounds.

**Accent Colors (For shapes, highlights, and conceptual elements):**
- `#e66a50` (Vibrant coral / Soft vermilion) - Primary highlight for active states, warnings, or key actions.
- `#2b4257` (Deep navy blue) - Primary structural color for text, main arrows, and solid foundations.
- `#5b8a72` (Muted emerald green) - Used for success states, secondary flows, or safe zones.
- `#7aa4c0` (Soft cerulean) - Used for tertiary accents, background elements, or neutral data points.

## Prompt Injection Template

When an Agent needs to inject this brand into the `{brand_guidelines}` placeholder, use the following descriptive text:

```text
professional corporate color palette, background in warm soft cream #fbfaf8 and light gray #f4f5f7, structural elements in deep navy blue #2b4257, accented with vibrant coral #e66a50, muted emerald green #5b8a72, and soft cerulean #7aa4c0, clean modern SaaS aesthetic, highly structured, clear visual hierarchy, soft subtle shadows
```

## Usage Rules
- Keep lines clean and geometric.
- Colors should be applied as flat fills or with very subtle, smooth gradients.
- Avoid cluttered backgrounds; focus purely on the structural elements and nodes.
- Maintain a balance where the navy blue provides the skeleton, while coral and green provide the life/flow.
