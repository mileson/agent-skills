# Modern Flow Diagram Style

Use this style for system architecture diagrams, business flowcharts, data flow diagrams, and conceptual product illustrations. This is the default choice for diagrammatic content unless another specific brand/style is requested.

## Color Semantics (Semantic Palette)

Use colors to convey meaning across the flow:
- **Input / R&D / Source Phase**: Warm Orange / Coral (`#FF7D66` or `#FA664A`)
- **Processing / AI Core / Review Phase**: Calm Blue (`#4A90E2` or `#5A8CDB`)
- **Output / Product / Results Phase**: Eco Green (`#50B87B` or `#61A871`)
- **Backgrounds**: Very light tint of the primary colors for card backgrounds (e.g., `#FFF3F0` for Orange, `#F0F5FA` for Blue, `#F0F9F3` for Green), or plain off-white/gray (`#F8F9FA`).
- **Text / Stroke**: Dark charcoal (`#2D3436`) for strong readability.

## Layout & Shapes

- **Card Design**: Use rounded rectangles for nodes/modules.
  - Add a subtle drop shadow or use a dark thin stroke for definition.
  - Two-tone cards: A solid color header (e.g., matching the phase color) with a light/white body for content.
- **Connections & Arrows**:
  - Use thick, smooth lines or arrows (e.g., 2pt-3pt weight) with gradients or solid colors that match the source or target node.
  - Avoid messy overlapping lines; use orthogonal (elbow) routing if needed.
- **Hierarchy Structure**:
  - Use large containing boxes (grouping areas) with a dashed border and very light background to signify a cluster, platform, or organizational division (e.g., "AI Agent Hub").

## Iconography & Typography

- **Icons**: Place a relevant line-art or minimalist icon at the top-left of each card or next to the main title.
- **Typography**:
  - Module Titles: Bold, sans-serif (e.g., Segoe UI, Calibri, Helvetica), 18-20pt.
  - Descriptive text: Regular weight, 12-14pt, slightly muted color (`#636E72`).
