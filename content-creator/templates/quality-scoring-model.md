# 爆款内容多维度分析模型 v2.0

> 用于评估内容的爆款潜力，指导优化方向

---

## 评分体系

**总分150分，转换为10分制**

---

## 一、观感分析法（50分，权重33%）

### 1.1 好奇心（10分）

**评分标准**：

| 分数 | 类型 | 示例 |
|------|------|------|
| 9-10分 | 认知差距型 | 揭示反常识的真相 |
| 7-8分 | 悬念型 | 制造强烈的未知感 |
| 5-6分 | 新奇发现型 | 提供新的视角或方法 |
| 3-4分 | 常规型 | 提供常见信息 |
| 0-2分 | 无吸引力 | 缺少新意 |

**优化建议**：
- 在标题和开头制造认知差距
- 使用反问、悬念引发好奇
- 提出颠覆性观点

### 1.2 颠覆性（10分）

**评分标准**：

| 分数 | 类型 | 示例 |
|------|------|------|
| 9-10分 | 数据型颠覆 | 用数据推翻常识 |
| 7-8分 | 观念型颠覆 | 挑战主流观点 |
| 5-6分 | 场景型颠覆 | 意外的应用场景 |
| 3-4分 | 行业型颠覆 | 行业内的新做法 |
| 0-2分 | 无颠覆性 | 符合常识预期 |

**优化建议**：
- 寻找反常识的数据
- 提出不同的解决思路
- 展示意外的成功案例

### 1.3 技术力（10分）

**评分标准**：

| 分数 | 评估维度 |
|------|---------|
| 9-10分 | 前沿技术 + 高复杂度 + 强价值 |
| 7-8分 | 成熟技术 + 中复杂度 + 明确价值 |
| 5-6分 | 常见技术 + 低复杂度 + 有价值 |
| 3-4分 | 基础技术 + 简单实现 |
| 0-2分 | 无技术含量 |

**优化建议**：
- 展示技术创新点
- 说明实现难度和门槛
- 量化应用价值

### 1.4 新鲜感（10分）

**评分标准**：

| 分数 | 说明 |
|------|------|
| 9-10分 | 前所未见的内容或角度 |
| 7-8分 | 少见的内容，新颖角度 |
| 5-6分 | 常见内容，不同角度 |
| 3-4分 | 常见内容，常见角度 |
| 0-2分 | 陈旧内容，老套角度 |

**优化建议**：
- 更换全新的切入角度
- 增加意外元素
- 使用差异化案例

### 1.5 沉浸感（10分）

**评分标准**：

| 分数 | 说明 |
|------|------|
| 9-10分 | 强烈的代入感，让读者完全投入 |
| 7-8分 | 较好的代入感，能吸引注意力 |
| 5-6分 | 中等代入感，可以读下去 |
| 3-4分 | 弱代入感，容易分心 |
| 0-2分 | 无代入感，难以坚持读完 |

**优化建议**：
- 使用第一人称叙事
- 增加具体场景描述
- 添加情绪和感受

## 评级标准

### 总分转换

**150分 → 10分制**：总分 ÷ 15

### 评级定义

| 评级 | 分数区间 | 说明 | 建议 |
|------|---------|------|------|
| 革命性创新 | 9.0-10.0 | 具备爆款潜力，强烈推荐发布 | ✅ 直接发布 |
| 显著创新 | 7.0-8.9 | 优质内容，推荐发布 | ✅ 直接发布 |
| 有效创新 | 5.0-6.9 | 及格内容，建议优化 | ⚠️ 优化后发布 |
| 微小创新 | 3.0-4.9 | 平庸内容，必须优化 | ❌ 必须优化 |
| 缺乏创新 | 0-2.9 | 低质内容，不建议发布 | ❌ 重新创作 |

### 阈值

- **excellent**：8.0+（直接发布）
- **good**：6.0+（建议优化）
- **acceptable**：4.0+（必须优化）

## 使用说明

### 1. 评分时机

- **阶段5.1**：首次评分（评估初稿）
- **阶段5.4**：二次评分（评估优化版）

### 2. 评分流程

```
1. 读取文章内容
2. 逐项评分（观感/视角/场景/本质）
3. 计算总分并转换为10分制
4. 确定评级
5. 诊断核心问题
6. 提供改进建议
7. 生成评分报告
```

### 3. 优化策略

**高优先级问题**（<6分）：
- 必须全部修改
- 针对性改进
- 验证提升效果

**中优先级问题**（6-7分）：
- 建议修改
- 权衡成本收益
- 避免过度优化

**保持优势**（>=8分）：
- 不做改动
- 保持原有风格
