# 小红书敏感词库（2026年最新版）

> **适用场景**：AI Agent 内容创作参考、小红书运营人员自查工具
> 
> **更新时间**：2026年2月
> 
> **使用说明**：本词库汇总了小红书平台的各类敏感词、违禁词，AI Agent 在生成内容时应避免使用这些词汇，或使用替代表达方式。

---

## 一、基本分类

### 1. 绝对违禁词
直接触发审核机制，内容可能被限流或删除

### 2. 敏感词
可能引发争议或误导用户的词汇

### 3. 行业限制词
特定行业禁止使用的夸大或绝对化表述

### 4. 变体词
通过拼音、谐音、符号等规避审核的词汇（同样会被检测）

## 十七、使用建议

### 17.1 AI Agent 内容生成时
1. **预检测**：生成内容后，先进行敏感词检测
2. **替代表达**：遇到敏感词，使用本词库提供的替代建议
3. **真实性**：强调真实体验，避免夸大其词
4. **客观性**：使用"个人认为"、"仅供参考"等客观表述

### 17.2 内容优化原则
1. **真实分享**：基于真实体验，不虚构、不夸大
2. **客观描述**：避免绝对化表述，使用相对客观的词汇
3. **合规表达**：不涉及医疗、政治、色情等敏感领域
4. **正向引导**：内容积极向上，传递正能量

### 17.3 高风险场景
1. **美妆个护**：功效描述需特别谨慎
2. **健康养生**：避免医疗用语
3. **投资理财**：不做收益承诺
4. **教育培训**：不做通过率保证
