> **版本**: 1.0 | **更新日期**: 2026-02-15
>
> **适用场景**: 趣味话题、开发者日常、情绪共鸣、用户故事、意外发现分享

## 逐图详细规范

### 图 1 — 情绪封面（必需）🎨

**目的**: 用强烈的情绪/反差/悬念驱动点击

#### 内容定义

| 要素 | 说明 | 示例 |
|:-----|:-----|:-----|
| 情绪钩子 | 夸张/悬念/反差的标题 | "App怎么变污了😭" / "上线第一天就翻车了" |
| 情绪emoji | 增强情绪感染力 | 😭🤯😱💀🫠 |
| 悬念元素 | 让人想知道"怎么回事" | 局部遮挡/打码的截图 / 数据曲线图 |

#### 设计规范

| 属性 | 规范 |
|:-----|:-----|
| 尺寸 | 1080×1440（3:4 竖屏） |
| 构图 | 大面积情绪文字（占 50%）+ 悬念视觉（占 50%） |
| 主标题 | 40-56px 加粗，可用红色/亮色强调关键词 |
| 情绪表达 | 通过配色传递情绪：惊讶用亮黄、尴尬用粉红、紧张用红黑 |
| 悬念处理 | 截图关键部分打码/模糊，制造"点进去看"的冲动 |

#### AI 生图提示词

```
设计一张情绪化的小红书封面图，竖屏3:4比例（1080×1440像素）。

画面布局：
- 上半部分：大字标题"{情绪钩子文案}"，加粗，{情绪色}
  - 关键词"{强调词}"用{高亮色}底色突出
  - 旁边散落2-3个{情绪emoji}
- 下半部分：{悬念视觉描述}
  - {如"一张App Store搜索结果截图，关键搜索词被红色马赛克遮挡，只露出令人好奇的边缘"}
  - 截图边缘微微倾斜，带投影，增加"偷拍/爆料"的感觉

背景：{情绪对应色系，如"浅粉色#FFE4E1（尴尬情绪）"}
风格：夸张但有趣，让人忍不住想点进来看怎么回事
```

### 图 2 — 证据截图（必需）📱

**目的**: 用事实/数据/截图支撑故事，增加可信度

#### 内容定义

| 要素 | 说明 |
|:-----|:-----|
| 证据内容 | 支撑故事的实际截图/数据/现象 |
| 标注重点 | 箭头/红框标注关键信息 |
| 简短说明 | 1 行说明"这就是证据" |

#### 设计规范

| 属性 | 规范 |
|:-----|:-----|
| 尺寸 | 1080×1440（3:4 竖屏） |
| 构图 | 截图居中展示，上方可选标题 |
| 截图 | 真实截图，不过度美化（保持真实感） |
| 标注 | 红色箭头/圆圈指向关键信息 |
| 隐私处理 | 敏感信息打码（用户名、具体数字等） |

#### 截图指引

```
1. 获取支撑故事的截图（数据后台/搜索结果/用户评论等）
2. 关键信息用红色箭头或圆圈标注
3. 敏感信息用马赛克打码
4. 可选：顶部加一行说明文字 "👀 看到这个搜索词我人都傻了..."
```

---

### 图 3 — 反转/结局/号召（可选）🎨

**目的**: 完成叙事闭环，引导互动

#### 内容定义

| 要素 | 说明 |
|:-----|:-----|
| 故事结局 | 正面反转 / 问题解决 / 自嘲幽默 |
| 号召行动 | 引导评论/点赞/关注 |
| 产品植入（可选） | 自然引出产品名，方便搜索 |

#### AI 生图提示词

```
设计一张故事结局图，竖屏3:4比例（1080×1440像素）。

画面布局：
- 上部：{反转结局的视觉表现}
  - {如"一个开心的表情包或插画角色，旁边文字'所以姐妹们，这款App叫「{产品名}」，不是那个意思！'"}
- 中间：产品名称大字展示"「{产品名}」"，方便用户搜索
- 底部：互动引导
  "你们搜到什么奇怪的关键词了吗？评论区分享 😂"

背景：轻松活泼的色调（浅黄#FFFDE7 或浅绿#F0FFF0）
风格：幽默、轻松、有趣，让人想评论分享
```
