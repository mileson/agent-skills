# 小红书图文笔记配图模板库

> **用途**：为 content-creator 阶段3（写作剧本）提供小红书图文笔记的标准化配图策划模板。
>
> **加载时机**：当目标平台包含 `xhs` 时，阶段3 自动加载本索引，根据笔记类型匹配模板。

---

## 模板选择决策流

```
检测笔记类型（基于阶段1选题 + 阶段2大纲）：

├─ 新产品/工具首次推广？
│   → 模板A：产品首发推广型（template-a-product-launch.md）
│
├─ 教用户如何使用某个功能/步骤教学？
│   → 模板B：功能教程型（template-b-tutorial.md）
│
├─ 已有产品的功能更新/版本迭代？
│   → 模板C：版本更新型（template-c-update.md）
│
├─ 征集用户意见/AB方案投票？
│   → 模板D：互动投票型（template-d-poll.md）
│
├─ 趣味话题/开发者日常/情绪共鸣？
│   → 模板E：故事话题型（template-e-story.md）
│
└─ 以上都不匹配？
    → 默认使用模板A，或由 Agent 组合多个模板元素
```

## 通用设计规范

所有模板共享以下基础规范：

### 尺寸与比例

| 用途 | 推荐尺寸 | 比例 | 说明 |
|:-----|:---------|:-----|:-----|
| 封面图（竖屏） | 1080×1440 | 3:4 | 点击率最高，信息流中占位大 |
| 封面图（方形） | 1080×1080 | 1:1 | 通用性强，适合简洁设计 |
| 内页图（竖屏） | 1080×1440 | 3:4 | 与封面保持一致 |
| 拼图/对比图 | 1080×1440 | 3:4 | 左右或上下分割 |

### 配色原则

| 受众定位 | 推荐色系 | 色值参考 |
|:---------|:---------|:---------|
| 女性用户（美妆/生活） | 暖粉、奶白、浅紫 | `#FFB6C1` `#FFF5EE` `#E8D5E0` |
| 科技/工具类 | 深蓝、渐变紫、亮白 | `#1A1A2E` `#6C63FF` `#FFFFFF` |
| 通用（独立开发者） | 奶白底+科技蓝点缀 | `#F8F9FA` `#4A90D9` `#333333` |

### 字体建议

| 元素 | 推荐字体 | 字号 | 字重 |
|:-----|:---------|:-----|:-----|
| 封面主标题 | 思源黑体 / 阿里巴巴普惠体 | 48-72px | Bold / Heavy |
| 副标题/说明 | 思源黑体 | 28-36px | Regular / Medium |
| 标注文字 | 苹方 / PingFang SC | 24-32px | Medium |
| 英文/数字 | Inter / SF Pro | 同上 | Semi-Bold |

### 图片来源分类

每张图的规划中会标注图片来源类型，Agent 据此给出不同的指引：

| 来源类型 | 标识 | Agent 输出内容 |
|:---------|:-----|:--------------|
| `screenshot` | 📱 | App 截图指引 + 标注说明（箭头、高亮区域、文字标注位置） |
| `design` | 🎨 | AI 生图提示词（中文 + English） + 设计规范 |
| `composite` | 🧩 | 拼图/对比图指引 + 各部分内容说明 |
| `photo` | 📷 | 实拍建议（场景、光线、角度） |

## 扩展指南

### 新增模板

1. 在本目录创建 `template-{id}-{name}.md`
2. 按照现有模板结构编写（概述 → 图片序列 → 逐图详细规范）
3. 在本 README 的模板一览表中添加条目
4. 在决策流中添加匹配条件

### 模板版本管理

- 每个模板文件头部包含版本号
- 迭代优化时更新版本号和变更说明
- 保持向后兼容：新增字段不删除旧字段
