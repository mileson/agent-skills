# Security Policy

If you believe this skill or its packaging contains sensitive data, credential leakage, or unsafe instructions, please do not open a public issue with the raw secret or exploit details.

Please disclose responsibly through a private channel first.

When reporting, include:

- where the issue appears
- why it may be sensitive or unsafe
- how to reproduce it safely
- what should be removed or fixed

Please avoid posting:

- API keys
- tokens
- passwords
- private URLs
- internal hostnames
- personal contact details that were not intended for publication
