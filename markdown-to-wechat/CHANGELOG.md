# 更新日志

## v1.9.0 (2026-01-23)

### ⭐ 重大优化

- 🎯 **代码块微信兼容优化** ⭐⭐⭐
  - **核心改动**：从表格结构改为简单的 `<pre><code>` 块状结构
  - **问题解决**：修复复制到微信公众号后变成"一行行表格"的问题
  - **兼容性提升**：使用微信支持的 HTML 标签和结构
  
  **技术实现**：
  - 移除表格（`<table>`）结构
  - 使用 `<pre><code>` 标签包裹代码
  - 保留语法高亮（Pygments）
  - 保留标题栏显示语言类型
  - 支持水平滚动（`overflow-x: auto`）
  - 支持垂直滚动（`max-height: 500px`）
  - 使用 `white-space: pre` 保持格式
  
  **保留特性**：
  - ✅ 灰色配色（`#F7F8FA`）
  - ✅ 圆角边框
  - ✅ 阴影效果
  - ✅ 语法高亮
  - ✅ 标题栏
  
  **移除特性**：
  - ❌ 表格形式的行号显示（为了兼容性）

- 🎨 **标题序号斜体效果**
  - H2-H6 标题的序号（1、1.1、1.1.1 等）添加斜体样式
  - 使用 `font-style: italic;` 实现优雅的倾斜效果
  - H2 序号：增大 50% + 斜体 + 深青色
  - H3-H6 序号：正常大小 + 斜体 + 深青色

### 🔧 配置调整

```css
/* 代码块 pre 样式 */
background-color: #F7F8FA;
overflow-x: auto;
overflow-y: auto;
max-height: 500px;
white-space: pre;
word-wrap: normal;
padding: 12px 16px;
line-height: 1.6;

/* 标题序号样式 */
font-style: italic;
color: #0D9488;
```

- ✅ 代码内容可以正常复制

- ✅ 不会出现"一行行表格"现象

## v1.2.0 (2026-01-23)

### ✨ 新增功能

- 🎨 **淡橙色强调效果** ⭐
  - **加粗文本**（`**text**`）→ 淡橙色背景高亮效果
    - 类似荧光笔划线重点的视觉效果
    - 使用渐变背景（60% 透明 + 40% 淡橙色）
    - 适用于重点内容、关键数据
  
  - **删除线**（`~~text~~`）→ 橙色虚线下划线
    - 不再是删除效果，改为强调效果
    - 虚线样式（dashed）更柔和
    - 适用于对比数据、特别提示

### 🎨 配色新增

- `highlight_bg`: #FFF3E0（淡橙色背景）
- `highlight_color`: #FF9800（橙色）
- `highlight_border`: #FFB74D（淡橙边框）

### 📊 实际效果

```markdown
这是**重点内容**，这是 ~~对比数据~~。
```
↓ 转换后
- "重点内容"：淡橙色背景高亮
- "对比数据"：橙色虚线下划线

## v1.1.0 (2026-01-23)

### ✨ 新增功能

- 🎨 **新增深蓝色主题** (`deep-blue`)
  - 主题色更深（#1E6FBA），更专业稳重
  - 适合商务报告、正式内容
  - 代码高亮使用 monokai 风格
  - 标题字号更大（H1: 22px, H2: 19px）
  - 间距更宽松，更有呼吸感

### 🔧 优化改进

- ✅ 创建 `convert.sh` 便捷脚本
  - 自动检测并复用虚拟环境
  - 智能依赖管理，避免重复安装
  - 命令更简洁易用

- ✅ 完善文档系统
  - 新增 `THEMES.md` 主题使用指南
  - 新增 `OPTIMIZATION_NOTES.md` 优化说明
  - 更新 `SKILL.md` 添加主题选择说明
  - 更新 `README.md` 和 `QUICKSTART.md`

### 🎨 主题对比

| 特性 | 淡蓝色主题 | 深蓝色主题（新）|
|------|-----------|---------------|
| 主题色 | #55C9EA | #1E6FBA |
| 风格 | 清新活力 | 专业稳重 |
| 适用场景 | 技术文章 | 商务内容 |
| H1 字号 | 20px | 22px |
| 代码高亮 | friendly | monokai |

### 📊 使用方式

```bash
# 深蓝色主题（推荐）
./convert.sh article.md --theme deep-blue -o output.html

# 淡蓝色主题
./convert.sh article.md --theme light-blue -o output.html
```
