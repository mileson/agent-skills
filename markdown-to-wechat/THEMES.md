## 🔧 自定义主题

### 创建自己的主题

1. 复制现有主题：
```bash
cp -r templates/deep-blue templates/my-theme
```

2. 编辑配置文件：
```bash
vi templates/my-theme/theme.yaml
```

3. 修改颜色配置：
```yaml
colors:
  primary: "#YOUR_COLOR"        # 你的主题色
  text_primary: "#YOUR_TEXT"    # 你的正文色
  bg_code: "#YOUR_BG"           # 你的代码背景
```

4. 使用自定义主题：
```bash
./convert.sh article.md --theme my-theme -o output.html
```

### 推荐配色工具

- **Adobe Color**: https://color.adobe.com/
- **Coolors**: https://coolors.co/
- **Material Design Colors**: https://materialui.co/colors

### 配色原则

1. **对比度**: 确保文字与背景对比度 >= 4.5:1
2. **颜色数量**: 全文不超过 3 种主要颜色
3. **色彩心理**: 
   - 蓝色：专业、信任、稳定
   - 绿色：自然、成长、环保
   - 红色：热情、紧急、重要
   - 灰色：中性、专业、简洁

## 🆘 常见问题

### Q: 如何切换主题？

A: 使用 `--theme` 参数指定：
```bash
./convert.sh article.md --theme deep-blue -o output.html
```

### Q: 可以混用主题吗？

A: 不建议。每篇文章使用一个主题，保持风格统一。

### Q: 如何查看所有可用主题？

A: 查看 templates/ 目录：
```bash
ls templates/
```

### Q: 深蓝色主题比淡蓝色慢吗？

A: 不会。两个主题的转换速度完全一致。
